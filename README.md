## Prerequisites

Ensure you have the following installed.

```
- kubectl
- kubectx
- helm
```

# Usage

```bash
git clone https://github.com/iquzart/k8scripts.git -o ~/.bin

echo "export PATH=$PATH:~/.bin" > ~/.zshrc

source ~/.zshrc
```
